# Backend Authentication System

## Project Overview
This project is a backend authentication system developed using Node.js and JWT-based authentication. It provides secure user authentication with validation and token-based authorization.

The system handles:
- User registration
- User login
- JWT token generation
- Protected route authentication
- Cookie-based token storage
- Email, Username, and Password Validation

## Features
- User Signup and Login
- JWT-Based Authentication
- Cookie-Based Token Storage
- Protected Routes
- Secure Authentication System
- Backend API Handling
- Input Validation for Email, Username, and Password

## Technologies Used
- Node.js
- Express.js
- JavaScript
- JWT (JSON Web Token)

## How to Run

1. Install dependencies:

```bash
npm install
```

2. Start the server:

```bash
node server.js
```

3. Open the application in the browser or test APIs using Postman.

## Objective
To implement a secure backend authentication system using JWT-based authentication, cookie-based token storage, and input validation.
